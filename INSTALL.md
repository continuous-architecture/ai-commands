# Installation

## Option A — Terminal (recommandé)

```bash
# Extraire l'archive tar.gz
tar -xzf ai-commands-repo.tar.gz

# Copier les commandes dans votre projet
cp -r ai-commands/.claude/commands/ mon-projet/.claude/commands/
```

## Option B — Sans terminal

1. Les fichiers `.md` du dossier `commands/` sont les commandes
2. Dans votre projet, créez le dossier `.claude/commands/`
3. Copiez-y tous les fichiers `.md` de ce dossier `commands/`

Note : sur macOS, les dossiers commençant par `.` sont cachés par défaut.
Pour les voir dans le Finder : `Cmd + Shift + .`
